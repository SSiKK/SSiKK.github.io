Description

    Here's a description about our CS422 Project 2: Knockin' on Heaven's Door.
    The application is called Doorway: Your Door Your Way created by Kyle, Shiwangi, Krishna and Saumya.

    Doorway is a Interface Design of a door, both inside and outside which is laid out on a Canvas. You can think of the interface as a high-tech door. This application is a user interface demonstrating various functionalies that a door can have like locking/unlocking, camera to talk to people at your door, notifications, security for your home and other decorating features designed on a JavaScript Canvas. This page will provide you with the features of the Doorway application and instructions on how to use the application. So, let's get started!

Instructions

  • The following is a detailed instruction of how to use the Doorway application which includes:

    How to use it?

    State of the door (locked or unlocked)





    Initial state of the locking/unlocking mechanism: On the inside door, you have a Red bolt feedback suggesting that the door is locked. When a user from outside unlocks the door, a feedback in the form of a Green ring around the doorknob is shown and the bolt turns Green too. When the user from the outside locks the door, a feedback in the form of a Red ring is shown on the outside and the bolt turns Red too. This feedback ring stays only for few seconds and then disappears.

    Ability to lock or unlock from inside





    User can click on the bolt which can lock or unlock the door from inside. A feedback in the form of color change from Green -> Red and from Red -> Green is sent to the user for locking and unlocking respectively.

    Menu System



    The menu system encapsulates pretty much all the features of our User Interface. The menu icon is a grid like icon just below the door knob. The following are the menu options we have incorporated:

    • Language
    • Chimes
    • Child Safety
    • Keys
    • Home Alarm
    • Users
    • Camera
    • Mirror
    • Notes
    • Maps
    • Door Log
    • Tutorial
    • Emergency
    • Theme
    selecting an icon would open up the desired functionality. We have made sure that the icons are pretty descriptive but in case a user might get confused, we have added a label below the icons too. When a user is done using the function, we get back to the menu system again.

    Themes on the outside door from inside



    We have incorporated in our design themes for seasons like Halloween, Christmas, Birthdays and also traditional wooden, metallic themes. User can display the themes on outside the door from inside by selecting the Themes icon from the menu system.

    Themes on the inside door





    Similarly, you can select the theme for the inside door by selecting inside option or you can change on both doors.

    Ability to see outside the door life-size



    By clicking on the Camera option in the menu, you get a condensed view of the person standing outside the door. You can a life-size view by clicking on the expand button. You can toggle these two options.



    Turn on/off inside camera for see-through







    In our interface, you can Turn On the camera for the person outside to see you. The camera icon turns green when you make it see-through. You can make this a condensed view or a life-size view for the person outside. The feedback is given in terms of small map on the inside door where you can see if the person has a full-sized view or condensed view.

    Personalized information about appointments, events, To-do list



    In our design we have incorporated adding notes and events for yourself to have a look as and when you want to move outside the house. In our design, you can see messages left for you or if have any important events lined up.
    If you want to write a message on the inside door, we give you an option to send it to a specific user by using the saved profile or you can send it to a General user.

    Personalized information in the presence of multiple people



    Our interface supports the notifications for Multiple users as and when a general user approaches a door, you can show messages to them.

    Doorway is your mirror



    By clicking on the menu icon for Mirror, you can make your door act like a mirror.

    Weather forecast



    We have a Weather api running to show the Current weather outside with an icon and Temperature.

    Call police or fire in an Emergency







    When you click on the Emergency icon in the menu, you get a screen which takes you to Alert screen which in return asks you to either call Fire Brigade or Police when clicked on the alert icon. When you press on the buttons there you are basically Calling Police Department or Calling Fire Department.



    You can see it in Italian too!

    House alarm on/off



    Clicking on the home security sets the house alarm.

    Doorbell chime



    Part of the menu system is Chimes. When you click on the doorbell icon, chimes window appears. This has a checkbox control. You can select the desired chime and the next time somebody on your door presses a bell, you will have the desired chime for the doorbell.

    Ability to write message on inside door for General/Specific people



    Message can be sent to general or specific user from the inside door by writing to the scribble pad first and then clicking on the Note To button.



    This gives you an option to send the notes to specific or general user.

    Talk to people on the outside with an avatar



    You can choose to talk to people by clicking on the Avatar icon from the Camera View. You can toggle the screen to be of condensed view or life-size view.

    Ability to lock or unlock from outside with a key



    User profiles: Talking about the users, we have saved/can save user profiles with id, name, handprint, passcode and image information. So, if the user has a handprint and a passcode saved in the system, he/she can unlock the door.
    Locking/unlocking from the outside is done by using two mechanisms in our system:
    1. By Handprint: In our interface, we have placed a handprint area just besides the door knob at a reachable height. User from the outside if registered in the system, can unlock the door by placing their palm in the provided area.
    2. By passcode: Just besides the handprint area is a passcode area. User can press the button and enter the passcode from their phones. If the phone is registered with the interface, the user can unlock the door.


    On Authentication success, you will get a feedback of Green ring around the knob and you after a few seconds, the door automatically gets locked.

    'Push' the doorbell


    When the doorbell on the outside is pressed as user approaches the door, you can hear a 'ting' sound. There is a doorbell icon in our interface design right above the doorknob.

    Ability to write message on the outside to appear inside


    When there is no one in the house, the user gets a message and also the user can leave a message on Outside the door. A scribble pad is opened up to leave this message and the user can Send a note.

    Tutorials for the first-time users

    First time users get a Tutorial to set their account on clicking on the Tutorial icon from the menu. There is a series of steps that the user has to follow to get it all set up.
    1. Welcome screen greets the user in different languages and asks you to press in the hand icon.


    2. You can to choose from a set of languages on clicking on the desired flag icon.


    3. Right now we have a support for Hindi, English, Kannada, Spanish and Italian. All other flags default to either Spanish if you are a Spanish speaking country or English otherwise.

    4. Next, you can either take a profile picture or choose from an array of hip images to set your profile picture.

    5. Lastly, you can feed your handprint into the system by selecting the hand icon in the middle and it gives you a feedback by turning the icon into Green.

    6. You will have a modern style tutorial which sets the opacity low and highlights with a bunch of short yet crisp instructions on how to use the application.

    Additional features:

    Language support





    We have a support for 5 languages in our interface, you can choose an option to set to the desired language from the menu. We have a support for Hindi, English, Kannada, Spanish and Italian.

    Maps


    On clicking on the Maps icon in the menu, you can have a heavy Traffic location nearby when you are about to step outside.

    Time and Date


    We have Current Time and Date displayed on the top of the inside door.

    Child safety mechanism


    Note logs







    We have added a feature to take logs of the people who press your doorbell. We have a Time Stamp of the door bell being rung. If you press the doorbell, you can see the Time Stamp being added to the log.

    Three Tap mode

    Emergency mode is on as soon as a user does a three clicks or taps on the inside door. You can go a screen where you can directly make a call to the Police or Fire Brigade.
